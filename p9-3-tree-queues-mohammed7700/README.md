Benutzen Sie Add file → Upload files, um Ihre Dateien hochzuladen.

Beantworten Sie außerdem die Fragen in den Dateien

- [Frage1.md](Frage1.md)

- [Frage3.md](Frage3.md)

und halten Sie sich dabei an die Anleitung in den Dateien.
